# Changelog

本文档记录 AI Web Enhancer 的重要更新。

## [v2.5.0] - 2026-04-19

### 重构

- 重构 `AI-web-enhancer.js` 单文件内部结构，按站点适配、滚动导航、目录同步、UI 构建与启动入口进行分区
- 收敛重复的滚动容器、消息索引、目录节点获取逻辑，降低后续维护成本
- 标准化顶部 / 底部边界滚动配置，减少主流程中的匿名逻辑

### 修复

- 修复重构过程中遗留的编码问题，将脚本统一为 UTF-8 无 BOM
- 修复 `README.md` 与 `CHANGELOG.md` 中的乱码内容
- 继续稳定 ChatGPT / Gemini 下的目录点击、高亮同步、滚动跟随与边界导航逻辑

### 优化

- 优化 `scanContent()` 的采集、渲染与同步流程，使主路径更清晰
- 优化目录交互状态管理，减少点击锁定、手动高亮与滚动释放之间的重复判断

## [v2.4] - 2026-03-26

### 新增

- 为纯图片消息增加目录兜底标题，支持显示 `图片`、`图片 xN` 与 `图片：描述`
- 新增本地性能回归页 `stress-test.html`，用于验证长对话与大 DOM 场景

### 修复

- 修复 ChatGPT 中点击目录后页面跳转正确但高亮偶尔停留在上一条的问题
- 修复 Gemini 中点击目录后更容易落到 AI 回答区域而不是对应提问位置的问题
- 修复返回顶部过程中目录最后一次不同步的问题

### 优化

- 优化长对话与大体量 DOM 场景下的目录扫描、位置缓存与滚动同步性能
- 优化 `MutationObserver` 触发范围，减少无关 DOM 变化导致的重复扫描
- 优化目录跟随当前阅读位置的自动滚动体验

## [v2.3] - 2026-03-25

### 修复

- 修复 ChatGPT 中浏览 AI 长回复时目录高亮偶尔跳到最后一条问题
- 修复 Gemini 中点击回到顶部后页面与目录不同步的问题

### 优化

- 优化目录当前项识别逻辑
- 优化目录与页面滚动的同步体验
- 提升长对话场景下的目录定位稳定性

- 提升长对话场景下的稳定性
